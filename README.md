### Stow

- [GNU Stow](https://www.gnu.org/software/stow/) 

Use Stow to install the configs. (Need to place this config repo in the home directory)

- Arch: Run `sudo pacman -S stow`
- Homebrew: Run `brew install stow`
- Others: do search
